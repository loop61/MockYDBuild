​In Mod Design 1: Adventure Maps Kids Learn:​

Foundational Coding Concepts

- Kids use a command-line interface to code their Adventure Maps, drawing on core coding concepts along the way. In order to add all the features they want, they’ll declare and manage variables, create conditional if-then and if-else statements, implement logical operators and boolean flags, and pass appropriate parameters to a variety of functions.

Think like a Programmer

- ​As kids string together Command Blocks in order to add new features, they’ll need to conquer all of the same difficulties as a professional programmer. They’ll use computational and critical thinking to achieve the correct logical flow, reference code documentation to understand required parameters, and use standard debugging techniques when their game is not working as expected.

MCEdit, a Powerful 3D Modeling Tool

- ​​Rather than building their games block-by-block in Minecraft®, kids will use a 3D Modeling tool that allows them to design a beautiful world efficiently. MCEdit shares some powerful functionality with professional 3D Modeling programs that allows them to quickly transform and manipulate objects in a 3-dimensional space to achieve their desired result.

How to make a Pro-Level Adventure Map

- Kids learn how to make an Adventure Map that looks and plays just like the ones they have seen online. They’ll use the same techniques as professional Minecraft® map-makers to create an incredible game that they can share with family and friends.


Mod Design Adventure Maps Requires:

  • Students must complete Mod Design 1 before taking Mod Design 1: Adventure Maps
  • Windows 10, 8,7, or Vista, or Mac: OS 10.8 or higher, 64-bit (after 2008)
  • Chrome, Firefox or Safari web browser
  • Network speeds greater than 1mbps
  • Minecraft® for Mac or PC (not pocket edition, or Windows 10 edition) on your computer.
  • Computers must meet the hardware requirements of Minecraft®.
